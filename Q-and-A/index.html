    How to require hkuportal for admin login

    1. Edit html/theorigo/application/config/theorigo.php, go to section ### Config for LDAP
    2. Uncomment define('CAS_LOGIN', 'defined'); to use hkuportal login.
    3. Set host to hkuportal.hku.hk
    4. Comment out # define('CAS_LOGIN', 'defined'); to turn off hkuportal login.
    Edit html/theorigo/application/config/theorigo.php to use hkuportal login for admin
    ### Config for LDAP
define('CAS_LOGIN', 'defined'); /* Uncomment to turn on admin login with hkuportal - Brian added */
$config['cas_login'] = array(
                            'host' => 'hkuportal.hku.hk', /* Brian changed to use hkuportal */
                            'port' => '443',
                            'uri' => '/cas',
                            'change_session_id' => false
                        );

    Make folder writable by Apache

    The ~/html/writables folder in School server needed to be writable by Apache. Use setfacl command to change access control of these folders so Apache can write to these folder. User (sbms) can delete files created by Apache.
