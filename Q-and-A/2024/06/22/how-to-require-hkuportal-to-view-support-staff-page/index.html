    How to require hkuportal to view support staff page

    Tip

    Ensure user has a hkuportal UID (HKU member) before accessing the page.

    1. copy /html/theorigo/application/modules/staff/controllers/Staff.php as Staff-original.php
    2. Replace Staff.php with the following: 
      <?php if (!defined('BASEPATH')) exit('No direct script access allowed');
#Brian modified to include HKU portal login#

$page_url = $_SERVER['REQUEST_URI'];

$page_require_portal = "/support-staff|school-manager|finance-hr|ke-gss|research-innovation|tlbdp|lab-support|anatomy-education|tlbdp/";
if (preg_match($page_require_portal, $page_url)) {
    //require_once(__DIR__ . "/phpCAS-1.4.0/vendor/autoload.php");
    require_once(APPPATH . '/libraries/CAS.php');
    phpCAS::client(CAS_VERSION_2_0, 'hkuportal.hku.hk', 443, '/cas', false);
    phpCAS::setNoCasServerValidation();     // Not recommended in production environment
    if (phpCAS::isAuthenticated()) {
        include('Staff-original.php');
    } else {
        phpCAS::forceAuthentication();
    }
} else {
    include('Staff-original.php');
}

    Warning

    Make sure phpCAS work with hkuportal.
