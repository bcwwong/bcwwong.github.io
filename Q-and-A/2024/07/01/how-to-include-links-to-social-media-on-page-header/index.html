    How to include links to social media on page header

    Elements in the web page header is described in html/theorigo/application/views/themes/common/header.twig. Edit this file to edit/add/delete elements in the page header.

    1. Edit html/theorigo/application/views/themes/common/header.twig to add social media icons; change "HKU Home" to "HKU"; change "Faculty Home" to "HKUMed";:
      edit html/theorigo/application/views/themes/common/header.twig to add social media links and change 'HKU Home' to 'HKU' and 'Faculty Home' to 'HKUMed'
      <div class="menu-top">
    <ul class="menu-top__list">

        <!-- Brian added -->
        <li class="menu-top__item link twitter">
            <a href="https://x.com/hkusbms/" target="_blank">{{ svg_include('twitter') | raw }}</a>
        </li>
        <li class="menu-top__item link instagram">
            <a href="https://www.instagram.com/hkusbms/" target="_blank">{{ svg_include('instagram') | raw }}</a>
        </li>
        <li class="menu-top__item link facebook">
            <a href="https://www.facebook.com/hkusbms" target="_blank">{{ svg_include('facebook') | raw }}</a>
        </li>
        <!-- end Brian added -->

        <li class="menu-top__item link">
            <a href="http://hku.hk" target="_blank">{{ _('HKU')  }}</a> <!-- change HKU Home to HKU -->
        </li>
        <li class="menu-top__item link">
            <a href="http://med.hku.hk" target="_blank">{{ _('HKUMed')  }}</a> <!-- change Faculty Home to HKUMed -->
        </li>
    2. Edit html/theorigo/application/views/themes/common/header.twig to modify color changes when hover over social media icons. Add the following lines before the page header. Find fill color and fill-opacity by trial and error. Change css to reduce spacing between icons (20px when window size is larger than 1599px and 10px when window size is less than 1599px);
      edit html/theorigo/application/views/themes/common/header.twig to modify hover color change
      <!-- Brian added -->
<style>
    .facebook:hover svg path,.menu-top__item:focus svg path {
            /* fill: #3B579D; */
            fill: #3B579D;
            fill-opacity: 0.6;
    }
    .instagram:hover svg path,.menu-top__item:focus svg path {
            /* fill: #c837ab; */
            fill: #3B579D;
            fill-opacity: 0.3;
    }
    .twitter:hover svg path,.menu-top__item:focus svg path {
            fill: #3B579D;
            fill-opacity: 0.5;
    }
    .linkedin:hover svg path,.menu-top__item:focus svg path {
            fill: #3B579D;
            fill-opacity: 0.5;
    }
    .menu-top__item:nth-child(n+1) {
        margin-left: 20px;
    } /* change spacing between icons */

    @media screen and (max-width: 1599px) {
        .menu-top__item:nth-child(n+1) {
            margin-left: 10px; /* spacing between social media icons changed to 10px when window size is less than 1599px */
        }
    }
</style>
<!-- End Brian added -->

      Where to get SVG codes?

      • Find and download SVG codes from UXWing.com:
        • https://uxwing.com/x-social-media-logo-icon/
        • https://uxwing.com/facebook-round-color-icon/
        • https://uxwing.com/ig-instagram-icon/
      • Add dimensions: add width="28" height="28" before viewBox="min-x min-y width height to specify 28x28 pixels icons. Other dimension may also work, find best looking combinations with trial and error.
