    How to use intraweb.hku.hk

    You can put pages in intraweb server to limit access to these pages.

    • Server: intraweb.hku.hk
    • Username/password: same as that in www.sbms.hku.hk
    • Available directories:
      • /web/docs/local/sbms/ (default login directory; content accessible by computers in campus network)
      • /web/docs/reserved_1/sbms/ (files accessible to users with valid hkuportal account)

    Note

    • Files put in directory /web/docs/local/sbms/ will be accessible by computers in campus network. No portal login is required.

    • Page URL format: https://intraweb.hku.hk/local/sbms/[sub-directory]/[filename] e.g., https://intraweb.hku.hk/local/sbms/SYChan-bioc4999.pdf

    Note

    Files put in directory /web/docs/reserved_1/sbms/ will be accessible after hkuportal login.

    Page URL format: https://intraweb.hku.hk/intraweb_1/sbms/[sub-directory]/[filename] e.g., https://intraweb.hku.hk/reserved_1/sbms/bbms-fyp/test.php

    • Anyone with a valid portal account can access the file.

    • You can limit access to selected staff or studnets by including an access control list ("acl.txt") in the same directory as the files to be served. List the staff hkuportal IDs or student UIDs, one per each line in acl.txt file. Access control will be enforced on all files within the same directory. Example of the acl.txt file: 

      14803
104431
120045
116715
3035792705
      users with UID 14803, 104431, 120045, 116715, 3035792705 will be able to access files in the sub-directory where acl.txt is located.

    • You can create multiple sub-directories each with a differentaccess control list ("acl.txt") to tailor for different group of people.
